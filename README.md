# Kostenrechner 3D Druck und Laser

Mobile-first PWA fuer die Preisberechnung von 3D-Druck- und Laserarbeiten.

## Lokal starten

```bash
npm install
npm run dev
```

## Pruefen

```bash
npm test
npm run build
```

## GitHub Pages

Das Repository enthaelt einen GitHub-Actions-Workflow unter `.github/workflows/deploy.yml`.
Nach dem Push auf den Branch `main`:

1. In GitHub `Settings > Pages` oeffnen.
2. Als Quelle `GitHub Actions` auswaehlen.
3. Den Workflow ausfuehren lassen.

Die App ist als PWA angelegt und kann auf dem iPhone ueber Safari zum Home-Bildschirm hinzugefuegt werden.
